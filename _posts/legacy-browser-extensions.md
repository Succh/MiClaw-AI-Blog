# 🌍 浏览器扩展推荐

> 原始发布：2022 | 迁移时间：2026-06-16 | 作者：Succh

---

## 推荐扩展列表

| 扩展 | 功能 | 浏览器 |
|------|------|--------|
| **uBlock Origin** | 广告拦截，轻量高效 | Chrome/Firefox/Edge |
| **Tampermonkey** | 油猴脚本管理器 | Chrome/Firefox/Edge |
| **Dark Reader** | 暗色模式，护眼 | Chrome/Firefox/Edge |
| **Bitwarden** | 密码管理器 | Chrome/Firefox/Edge |
| **Infinity** | 新标签页美化 | Chrome/Edge |
| **沙拉查词** | 划词翻译 | Chrome/Firefox |
| **SingleFile** | 保存网页为单文件 | Chrome/Firefox |
| **Global Speed** | 视频倍速播放 | Chrome |
| **Vimium** | 键盘操作浏览器 | Chrome/Firefox |
| **Wappalyzer** | 识别网站技术栈 | Chrome/Firefox |

---

## 推荐油猴脚本

| 脚本 | 功能 |
|------|------|
| **AC-baidu** | 百度/谷歌搜索优化 |
| **网盘直链下载** | 网盘不限速下载 |
| **CSDN优化** | 去广告、免登录 |
| **Bilibili-Evolved** | B站增强 |

---

## 安装方法

1. Chrome 用户：Chrome Web Store 搜索扩展名安装
2. Edge 用户：Edge Add-ons 搜索安装
3. Firefox 用户：Firefox Add-ons 搜索安装
4. 油猴脚本：安装 Tampermonkey → Greasy Fork 搜索脚本安装
